The data were retrieved on June 8, 2016.

## Variables

| variable       | description                                 |
|----------------|---------------------------------------------|
| land           | national team                               |
| land_id        | id of the national team on transfermarkt.de |
| name           | name of player                              |
| position       | position                                    |
| verein         | team                                        |
| verein_id      | team id                                     |
| gebdatum       | date of birth                               |
| alter          | age                                         |
| marktwert      | market value                                |
| liga           | league of the player                        |
